LeetCode 23
[English](https://leetcode.com/problems/merge-k-sorted-lists/)
[中文](https://leetcode-cn.com/problems/merge-k-sorted-lists/)

## 题目
合并 k 个排序链表，返回合并后的排序链表。请分析和描述算法的复杂度。

示例:
```
输入:
[
  1->4->5,
  1->3->4,
  2->6
]
输出: 1->1->2->3->4->4->5->6
```
